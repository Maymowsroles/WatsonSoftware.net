What it does

Better cursors, zero frustration

Installing a custom cursor theme in Windows normally involves extracting files, navigating to Mouse settings, browsing for a folder, and setting each cursor state individually. CursorKit turns that into: open app, click theme, done.

It ships with 30 ready-to-use themes ranging from clean minimal sets to colorful ones. You can also import any INF-based cursor pack you've downloaded from the web. The per-app override feature lets you keep a different cursor in, say, your drawing software.

  • ✓30 built-in cursor themes, ready to apply immediately
  • ✓Live preview — hover over a theme to see it in action
  • ✓One-click apply, one-click revert to default
  • ✓Import any .INF cursor pack from the web
  • ✓Per-application cursor overrides
  • ✓Works alongside custom pointer speeds and trails
  • ✓Backs up your current cursor setup before any change
  • ✓Tiny installer, no registry mess left behind if uninstalled
